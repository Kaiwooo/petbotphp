+## Установка на Render.com
+
+1. Клонируйте репозиторий:
+   ```bash
+   git clone https://github.com/Kaiwooo/petbotphp.git
+   cd petbotphp
+   ```
+2. Создайте новый **Web Service** на Render.
+3. Укажите команду запуска:
+   ```bash
+   php -S 0.0.0.0:$PORT bot.php

+4. При необходимости добавьте переменную окружения `APP_DEBUG=1`.
